# Калькулятор виживання в ІТ

Streamlit-застосунок з двома вкладками, побудований на реальному датасеті
зарплат DOU (грудень 2025), ~12 400 респондентів.

**Демо:** https://it-survival-calculator.streamlit.app/

## Файли

| Файл | Призначення |
|---|---|
| `data_utils.py` | Очищення даних: парсинг стажу, групування тайтлів/категорій/мов, формула індексу сивини |
| `model.py` | Навчання моделей, збереження `.joblib` |
| `app.py` | Streamlit-застосунок |
| `.streamlit/config.toml` | Кольорова тема |
| `dou_raw.csv` | Сирий датасет DOU (грудень 2025) |
| `requirements.txt` | Залежності |

## Запуск

```bash
pip install -r requirements.txt
python model.py
streamlit run app.py
```

## Вкладка "Індекс сивини"

Проста лінійна регресія: X — роки ІТ-стажу, y — індекс сивини (умовна
метрика, порахована з віку та стажу за формулою в `data_utils.py`).
Графік показує прогноз моделі поруч із найближчим реальним респондентом
із того самого датасету.

## Вкладка "Оптимізація виживання"

1. Багатофакторна лінійна регресія прогнозує зарплату за стажем, віком,
   тайтлом, категорією, рівнем англійської та мовою програмування.
2. Лінійне програмування (`scipy.optimize.linprog`) розподіляє бюджет
   виживання між кавою, терапією, чаєм і заощадженнями так, щоб
   максимізувати "продуктивність" при заданому бюджеті.
3. Слайдери особистих пріоритетів дозволяють змістити вибір моделі
   (наприклад, підвищити пріоритет терапії відносно кави).

## Очищення даних

- Стаж — вільний текст ("15-20 років", "Пів року") парситься в роки
- Тайтли згорнуто з 15 до 11 рівнів сеньйорності
- Категорії — семантично близькі варіанти об'єднано (напр. три типи
  підтримки → "Support")
- Мови програмування — топ-14 залишено, решта → "Інша"

## Відомі обмеження

- Лінійна регресія погано екстраполює для рідкісних комбінацій ознак
- Модель використовує узагальнену категорію (напр. "Analyst"), а не
  вузьку посаду — для конкретних ніш офіційний калькулятор DOU може
  давати точніше значення на малій вибірці анкет
- Індекс сивини — умовна метрика, не науковий показник
